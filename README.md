Данная модель блокчейна построена на стеке PostgreSQL,FastAPI. Для запуска сервера исползуется uvicorn на 8000 порту.
systemctl start postgresql.service

1)
Для запуска необходимо установить Базу данных PostgreSQL и создать для неё пользователя.
В файле app.py в этой строке замените {name} на имя своего ползователя:
connect = psycopg2.connect("dbname=BChain user={name}")

2) Для пользования доступны три ссылки
http://0.0.0.0:8000/ -главная страница со всеми блоками
http://0.0.0.0:8000/generate -после перехода запустится генерация блоков
http://0.0.0.0:8000/mine -запишите данные и имя пользователя в блок
Также доступны вторичные ссылки для просмотра данных
